## ✨ الميزات

- 💰 تحويل العملات (أكثر من 50 عملة)
- ⚖️ تحويل الوزن (كيلو، رطل، أوقية ذهب، مثقال)
- 📏 تحويل الطول (متر، قدم، ذراع، سنة ضوئية)
- 🌡️ تحويل الحرارة (سيلزيوس، فهرنهايت، كلفن)
- 📅 تحويل التاريخ (هجري ↔ ميلادي)
- 🏺 المقاييس التراثية (ذراع، شبر، من، مثقال)
